# 📸 Diary in Frames | Premium Visual Portfolio

A high-end, cinematic photography and videography portfolio website built with a minimalist, luxury aesthetic. Features an edge-to-edge crossfade cinematic hero arena, a dynamic responsive masonry layout, a custom premium glassmorphism user interface, fluid inertial scrolling mechanics, and an interactive 3D depth-shifting perspective engine.

---

## ✨ Features & Architecture

* **Premium Editorial Typography:** Integrated Google Fonts workspace blending structured, high-end sans-serif UI layers (`Plus Jakarta Sans`) with gorgeous, classic serif-italic accent lines (`Playfair Display`).
* **3D Perspective Interaction:** Implements an advanced CSS 3D transform matrix pipeline (`preserve-3d`). Hovering over exhibition panels skews them dynamically on a 3D plane, shifts layout elevation along the Z-axis, and casts deep ambient shadow maps.
* **Cinematic Crossfade Engine:** An automated multi-slide header array utilizing smooth opacity gradients to blend focal background frames seamlessly without hurting performance.
* **Inertial Kinetic Scroll Pipeline:** A lightweight, pure JavaScript linear interpolation (`lerp`) engine that captures native scroll physics and dampens container displacement for an ultra-smooth, premium trackpad feel.
* **Frosted Glassmorphism UI Components:** Fixed-position navigation structures and content containers styled with multi-layered back-drop filters (`backdrop-filter: blur(25px)`) for a clean, luxury digital finish.
* **Asynchronous Content Mapping:** Seamlessly loops through local image repository paths to auto-generate asymmetrical masonry modules, featuring beautiful native mountain landscapes, street portraiture, and cultural captures.
* **Immersive Lightbox Overlay:** Full-viewport interactive image modal workspace with dark ambient blur saturation layers for isolated artwork presentation.
