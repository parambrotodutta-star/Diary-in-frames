# Diary in Frames | Premium Visual Canvas

A high-end, responsive minimalist digital exhibition space dedicated to cinematic storytelling, atmospheric streetscapes, and timeless mountain vistas. Built with an avant-garde editorial aesthetic featuring dynamic glassmorphic panels, rich fluid background layers, and automated presentation engines.

## ✨ Core Design Features

* **Fluid Aurora Glaze Canvas**: An interactive backdrop built with shifting, saturated ambient blur elements and a high-definition film grain overlay to prevent harsh flat banding.
* **Asymmetric Exhibition Grid**: An editorial layout masonry configuration that gracefully displays original photographic frames without stretching or distortion.
* **Mobile-Responsive Engineering**: Full cross-platform scaling constraints that automatically drop fixed desktop boundaries on smaller screens to eliminate page cutting and image side-cropping.
* **Immersive Media Modal Layer**: High-performance interactive architecture supporting dynamic view expansions for high-resolution content inspection.

## 🛠️ Built With

* **HTML5** - Semantic structural mapping and responsive viewport configuration.
* **CSS3** - Custom variables (`:root`), advanced aspect-ratio handling, and dynamic ambient animation engines (`@keyframes`).
* **JavaScript** - Automation routing for interactive image arrays and view controllers.
