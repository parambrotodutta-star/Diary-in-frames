# 📸 Diary in Frames | Warm Aurora Glass Canvas

A premium, highly vibrant visual portfolio for displaying cinematography and photography work. Features an avant-garde asymmetric 3D tilt-grid structure floating over a dynamic liquid aurora moving backdrop textured with cinematic analog film noise.

---

## 🌟 Refined UI Engineering & Features

* **Vibrant Aurora Canvas Texture:** Replaces flat white/dark tones with a colorful moving blend of deep sunset coral, gentle amber, and aurora teal gradients layered under a rich, textured procedural film-grain SVG texture overlay.
* **Flawless Container Length Fixes:** Eliminates mathematical scrolling layout tracking overrides completely. Layout structures rely on native CSS flow parameters, removing empty whitespace bugs or layout offsets below the studio footer.
* **Simplified Asset Name Configurations:** Streamlines image references down to robust paths (`mountain-night.png` and `prayer-flags.png`) to cleanly bypass case-sensitivity rendering bugs on live deployments.
* **3D Interactive Vector Tracking:** Calculates cursor location relative to a card's physical center in real time, translating perspective matrixes in modern CSS natively.
* **High-Saturate Editorial Glassmorphism:** Uses deep backdrop blurs (`blur(25px) saturate(160%)`) bound by custom high-light stroke configurations to create beautiful layout layers.
