# 📸 Diary in Frames | Minimalist Art-Gallery Portfolio

A high-end, bright editorial cinematography and photography exhibition platform. The workspace features an asymmetrical 3D tilt-grid layout layered over an ultra-clean platinum background canvas with dynamic cursor tracking and subtle procedural film-grain textures.

---

## 🌟 Refined Light Features & Redesign Architecture

* **Gallery-White Studio Canvas:** Moves completely away from deep dark tones to an ultra-clean, minimalist off-white (`#f3f4f7`) and pure white card layout. This color space mirrors premium modern art exhibitions, emphasizing your photography color grading.
* **Flawless Spacing Fixes:** Resolves calculation conflicts caused by traditional heavy linear-interpolation kinetic wrappers. The site length relies strictly on clean, native, hardware-accelerated CSS properties, eliminating awkward dead space or scrolling voids beneath the footer.
* **Avant-Garde Asymmetrical Grid:** Breaks from standard box templates into an elegant, offset presentation structure reminiscent of elite fashion houses and design boutique agencies.
* **3D Cursor-Tracking Interactive Rig:** Implements real-time mathematical translation using JavaScript vectors. Moving your mouse over any exhibit panel computes relative coordinates from its center, tilting the element seamlessly on a true 3D CSS matrix.
* **Apple Light Glassmorphism:** Structural panels utilize light glassmorphic specs (`blur(25px) saturate(150%)`) surrounded by soft, low-opacity strokes (`rgba(0,0,0,0.07)`) to form a crisp layout with subtle depth over ambient background lighting accents.
