# 📸 Diary in Frames | Premium Editorial Canvas

An avant-garde, luxury-tier cinematography and photography exhibition platform. This codebase has been completely rebuilt from the ground up, discarding rigid flat grids in favor of an asymmetric 3D layout wrapped in deep frosted glassmorphism, dynamic cursor tracking, and an organic mesh gradient system with a custom film-grain noise texture overlay.

---

## 🌟 Advanced Features & Redesign Architecture

* **High-Contrast Premium Textures:** Ditches standard flat backgrounds for an ultra-premium background matrix. Features smooth, slow-moving ambient light orbs over a multi-layered color field combined with a custom procedural SVG noise-grain overlay to mimic analog film textures.
* **Avant-Garde Asymmetrical Grid:** Built using a luxury editorial structural layout. Project cards break away from blocky, standard masonry patterns into an offset presentation strategy reminiscent of high-end design agencies and fashion houses.
* **3D Cursor-Tracking Interactive Rig:** Implements an advanced real-time mathematical translation engine in JavaScript. Moving the mouse over any showcase panel computes relative coordinate vectors from the element's geometric center, dynamically tilting the glass container on a true CSS 3D matrix.
* **Extreme Back-Drop Saturation & Frost:** Navigation bars and display surfaces utilize maximum glassmorphism parameters (`blur(35px) saturate(200%)`) flanked by crisp dual-stroke outer borders to maximize layout depth over moving canvas backgrounds.
* **Apple Inertial Kinetic Scroll Pipeline:** Features a high-performance linear interpolation (`lerp`) scrolling container that captures and dampens trackpad inputs for an ultra-premium, fluid user experience.
